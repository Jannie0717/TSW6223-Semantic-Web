Semantic Course Recommendation System

Solution 1 website for education-domain course recommendation.

System Purpose

This website helps students find suitable courses based on their selected learning interest area. The Current Skills checkbox section and Career Path / Skill Gap Analysis section have been removed, so this version focuses only on Course Recommendation.
